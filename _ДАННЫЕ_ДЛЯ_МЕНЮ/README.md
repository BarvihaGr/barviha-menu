# Папка для данных меню

Сюда складывайте всё что нужно добавить или поменять в меню Барвиха.
Когда положите файлы — напишите в чате что добавили («новые фото в Кухню»,
«меняю цены на кальяны», «сезонное меню готово»), и я подхвачу.

## Структура

```
_ДАННЫЕ_ДЛЯ_МЕНЮ/
├── 01_БЛЮДА/         ← тексты блюд (название, описание, цена, состав, КБЖУ)
│   ├── Кальяны/
│   ├── Бар/
│   ├── Кухня/
│   └── Десерты/
├── 02_ФОТО/          ← фотографии блюд
│   ├── Кальяны/
│   ├── Бар/
│   ├── Кухня/
│   └── Десерты/
├── 03_СЕЗОННОЕ_МЕНЮ/ ← блюда которые есть только в сезон
│   ├── весна-лето/
│   └── осень-зима/
├── 04_ЛОКАЦИИ/       ← данные по конкретным заведениям (адрес, фичи, VIP)
└── 05_КАРАОКЕ/       ← инфо по караоке-сценам
```

## Куда что класть

### 📸 Фотки блюд → `02_ФОТО/<категория>/`
Имя файла — точно как блюдо, например:
- `Тартар из говядины.jpg`
- `Цезарь с креветкой.webp`
- `Pina Colada.png`

Формат — `.jpg`, `.png` или `.webp`. Лучше квадрат или 4:3, от 1200px.

### 📝 Текст блюда → `01_БЛЮДА/<категория>/`
Один файл `.md` на блюдо или один общий список — как удобнее.
Пример формата (см. шаблон в каждой подпапке).

### 🌱 Сезонное меню → `03_СЕЗОННОЕ_МЕНЮ/`
Если блюдо только зимой — кладёте в `осень-зима/`,
если только летом — в `весна-лето/`.

### 📍 Локации → `04_ЛОКАЦИИ/`
Один `.md` файл на каждое заведение (Менделеевская, Красная-площадь, и т.д.).

---

## Как сказать мне «обновись»

Просто напишите в чате одно из:
- «Добавил фотки в Кухню» — пройдусь по новой папке, подцеплю
- «Меняю цены в Кальянах» — открою файл, обновлю
- «Готово сезонное меню» — внесу
- Или просто пришлите файл прямо в чат — я положу куда надо.
