# Сезонное меню

Сюда кладёте блюда которые есть **только в сезон**.

## Когда что активно

- `весна-лето/` — март–август (свежие овощи, лёгкие коктейли, безалкогольное)
- `осень-зима/` — сентябрь–февраль (тыквенные супы, согревающие напитки)

Когда даёте старт сезону — пишете в чате: «запускаем весна-лето» —
я переключу флаги и сезонные блюда появятся в меню. По окончанию сезона —
«закрываем весна-лето» — и они скроются (но не удалятся, на следующий год вернутся).

## Формат файлов
Такой же как в `01_БЛЮДА/` — обычный `.md` с полями.
Дополнительно укажите в начале:

```
**Сезон:** весна-лето
**Активно с:** 1 марта
**Активно до:** 31 августа
```
