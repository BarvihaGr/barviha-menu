CANELA — как вставить (drop-in за 1 минуту)
============================================

Сейчас на макете живёт Raleway (тонкий сан-сериф без засечек) как fallback.
Как только сюда положишь файлы Canela — он автоматически встанет во все
заголовки и кнопки (Canela уже стоит первой в --font-display, см. globals.css).

ШАГ 1. Получить файл легально и бесплатно для макета:
  • fontstand.com/fonts/canela — бесплатный пробный период (триал), либо
    дешёвая помесячная аренда. Скачиваешь .otf/.woff2.
  • либо официальные TEST-шрифты на commercialtype.com/catalog/canela
    (кнопка Test / Pin to Test) — они и предназначены для макетов.

ШАГ 2. Положить файлы СЮДА (apps/menu/public/fonts/) с такими именами:
  • Canela-Light.woff2     (или .woff / .otf)   — вес 300
  • Canela-Regular.woff2   (или .woff / .otf)   — вес 400
  • Canela-Medium.woff2    (или .woff / .otf)   — вес 500
  (хватит даже одного Light — остальные необязательны)

ШАГ 3. Обновить страницу с Cmd+Shift+R. Всё.
Конвертер .otf -> .woff2 если нужно: transfonter.org (бесплатно, онлайн).
